# gbsip

本项目是 gbsip 的前端实现，基于 vue3 + antd-design-vue3 + typescript 实现，基于 [vue3-antd-admin](https://github.com/buqiyuan/vue3-antd-admin) 修改

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/vanjoge/gbSip
```

- 安装依赖

```bash
cd gbSip/ui

yarn install

```

- 运行

```bash
yarn serve
```

- 打包

```bash
yarn build
```
